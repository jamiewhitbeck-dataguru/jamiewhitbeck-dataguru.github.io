SaaS Infrastructure Modernization

When all the "professionals" said only enterprise-grade SSDs were acceptable in transactional database servers, I rejected the status quo, mitigated the risk, and built a server with consumer-grade SSDs.

By pairing consumer-grade SSDs with a battery-backed RAID controller, I delivered the same data durability while saving the company 43% on hardware costs. This decision set the infrastructure strategy for the company, ultimately saving over $35k on hardware costs for our modernization project, as well as reducing ongoing costs from power consumption and avoiding a costly data center expansion.

The result was a single server that replaced three legacy machines, which freed up critical data center space, delivered 400% performance improvement that customers immediately noticed, and provided a platform for significant scalability during a rapid-growth phase.

Scripted SaaS Database Migration

While most companies endure all-nighters and prayer circles for SQL migrations, I automated the entire process.

Using a reusable stored procedure deployed through our container registration database, I successfully migrated 500+ databases from three legacy servers to a single beast of a machine in under one hour—with less than 10 seconds of impact per tenant. The solution used dynamically-generated SQL to create and drop linked servers, control user access, execute DB backups and restores, and update container pointers—all while providing comprehensive logging.

User Data Encryption

During a self-initiated security audit as DBA for a CMMS vendor serving 1,000+ tenant databases and thousands of on-premises clients, I discovered that user credential "encryption" was laughably inadequate. With government agencies, hospitals, and critical infrastructure organizations using our platform—many leveraging our GIS module for sensitive asset mapping—this represented a catastrophic vulnerability waiting to happen.

I developed a robust, standards-compliant encryption solution that secured thousands of client credentials before this vulnerability could become front-page news.

How One Report Rewrite Uncovered a $4M Error

What started as a routine report migration from Excel to SSRS became a $4M discovery. During the rewrite, I uncovered a critical calculation flaw that had been systematically underreporting delayed shipments for two decades.

After proving the discrepancy across multiple systems, my analysis led to a multi-department effort that ultimately reduced backorders by approximately 50% ($7M) and improved customer satisfaction.

Transforming Overstock Reporting for Smarter Inventory Decisions

When a new WMS implementation and masterpack ordering changes created $5M+ in phantom overstock, I reengineered the legacy Overstock Report to account for seasonal demand patterns, masterpack quantities, and cross-system parameter misalignments.

The result was a strategic tool that empowered planners to distinguish real inventory problems from system artifacts and shift from firefighting to optimization.
